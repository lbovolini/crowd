## Build Project

```
mvn clean install
```

## Go to app folder

```
cd ~/app
```

## Run Hello

```
java -Dport=8006 -jar master-hello-1.0-SNAPSHOT.jar
```

## Run Client

```
java -jar crowd-1.0-SNAPSHOT.jar
```
